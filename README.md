# スリザーリンク Webアプリ

ブラウザ上で遊べるスリザーリンク（Slitherlink）のWebアプリケーションです。Next.js と Tailwind CSS を使用して構築されています。

## 実装されている仕様・機能

### 1. 盤面表示
- `8x8` マスのグリッドと、その周囲の点（ドット）を表示します。
- `文字列の配列` から盤面データをパースして表示する機能を実装しています。
  - スペースは「数字のないマス（空白）」、1〜3の数字は「ヒントとなる数字マス」として扱われます。

### 2. 操作方法（線の描画）
- **ホバー**: 点と点の間のスペースにマウスカーソルを当てると、カーソルがポインターに変わり、薄いグレーの線が浮かび上がります（当たり判定は16pxと太めに設定）。
- **左クリック（実線）**: 薄い線の領域を左クリックすると、黒い実線が引かれます。再度左クリックで元に戻ります。
- **右クリック（バツ印）**: 「ここには線を引かない」という推論を記録するため、右クリックで灰色の点線と中央に「×（バツ）」印を配置します。再度右クリックで元に戻ります（※デフォルトのコンテキストメニューは無効化されています）。

### 3. 解答補助機能
- **マスの背景色ハイライト**: マスの中央部分（数字や空白部分）をクリックすると、マスの背景が薄いピンク色に変わります。再度クリックすると元の透明な背景に戻ります。輪っかの内側や外側をメモするのに役立ちます。
- **数字の達成度チェック**: 数字の書かれたマスの周囲に引かれた実線の数が、その数字と完全に一致した場合、自動的に数字の背景に「薄い赤い丸」が表示されます。

### 4. 自動正解判定機能
ユーザーが実線を引いたり消したりするたびに、現在の盤面が正解かどうかを自動的にチェックします。以下の2つの条件を両方満たしたときに、「正解！」というポップアップ（アラート）が表示されます。

1. **すべての数字の条件達成**: 盤面上にあるすべての数字について、周囲に引かれている実線の数が数字と一致していること。
2. **1つの輪っかになっていること**: 引かれたすべての実線が、枝分かれも途切れもせず、全体でつながった「1つの輪っか」を形成していること（グラフ理論における各頂点の次数が2であり、連結成分が1つであることで判定）。

## 起動方法

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) にアクセスすると、パズルをプレイできます。
